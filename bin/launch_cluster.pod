=head1 DESCRIPTION

A tool for launching both single and multi node clusters of seqware. 

=head1 NAME

Cloud Bindle - SeqWare

=head1 REQUIRED

=item --config[=][ ]<cluster>

Specify the configuration file that Bindle will use to launch instances. 

=item --cluster[=][ ]<cluster>

specify the name of the cluster, found in your ~/.bindle/<platform>.cfg file, that has been previously specified.
